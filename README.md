# 🫀 CPR Education — ความรู้เล็ก ๆ ที่ช่วยชีวิตคนได้

เว็บไซต์ให้ความรู้เรื่อง CPR (การกู้ชีพ) ภาษาไทย พร้อมคู่มือทีละขั้นตอน โหมดฉุกเฉิน และแบบทดสอบความรู้

🔗 **Live Demo:** [kittiphongkubkub.github.io/CPR-IOT](https://kittiphongkubkub.github.io/CPR-IOT)

---

## ✨ ฟีเจอร์หลัก

| ฟีเจอร์ | รายละเอียด |
|--------|-----------|
| 📚 คู่มือ CPR ทีละขั้นตอน | คำอธิบายละเอียดพร้อมรูปภาพ |
| ⚠️ โหมดฉุกเฉิน | คำแนะนำแบบกระชับ + จังหวะเสียงช่วยกด |
| 📝 แบบทดสอบความรู้ | ทดสอบความเข้าใจของผู้ใช้ |
| 🎬 วิดีโอสาธิต | ดูตัวอย่างการทำ CPR จริง |
| 📱 Responsive Design | รองรับทุกขนาดหน้าจอ |

## 📁 โครงสร้างโปรเจก

```
CPR-IOT-main/
├── index.html              # หน้าหลัก
├── style.css               # สไตล์ทั้งหมด
├── script.js               # JavaScript (โหมดฉุกเฉิน, animation)
└── pages/
    ├── what-is-cpr.html    # CPR คืออะไร
    ├── steps.html          # ขั้นตอน CPR
    ├── who-needs-cpr.html  # CPR สำหรับใคร
    ├── dos-donts.html      # ควร/ไม่ควรทำ
    ├── resources.html      # สื่อการเรียนรู้
    ├── quiz.html           # แบบทดสอบ
    └── about.html          # เกี่ยวกับ
```

## 🛠️ Tech Stack

- **HTML5** — โครงสร้างเว็บไซต์
- **CSS3** — ออกแบบ UI ด้วย CSS Variables และ Animation
- **Vanilla JavaScript** — โหมดฉุกเฉิน, เสียงจังหวะ, Scroll Animation
- **Font Awesome 6** — ไอคอน

## 🚀 วิธีใช้งาน

1. Clone หรือดาวน์โหลดโปรเจก
2. เปิดไฟล์ `index.html` ในเบราว์เซอร์
3. ไม่ต้องติดตั้งอะไรเพิ่ม — ทำงานได้ทันที

## 📖 หน้าต่าง ๆ

- **หน้าหลัก** — ภาพรวมและสถิติสำคัญ
- **CPR คืออะไร** — ความรู้พื้นฐาน
- **ขั้นตอน CPR** — คู่มือทีละขั้นตอนแบบละเอียด
- **CPR สำหรับใคร** — กลุ่มเป้าหมายที่ต้องการ CPR
- **ควร/ไม่ควร** — ข้อควรระวัง
- **สื่อการเรียนรู้** — วิดีโอและทรัพยากรภายนอก
- **แบบทดสอบ** — ตรวจสอบความเข้าใจ

## ⚠️ ข้อจำกัดความรับผิดชอบ

เว็บไซต์นี้จัดทำเพื่อให้ความรู้เบื้องต้นเท่านั้น **ไม่สามารถทดแทนการฝึกอบรมจากผู้เชี่ยวชาญ** แนะนำให้เข้าร่วมหลักสูตรปฐมพยาบาลจากหน่วยงานที่เชื่อถือได้

> ข้อมูลอ้างอิงจาก WHO, สภากาชาดไทย, สถาบันการแพทย์ฉุกเฉินแห่งชาติ (NIEMS)
